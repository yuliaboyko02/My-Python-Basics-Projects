# 🐍 Python Fundamentals Project

## 📌 Overview
This project was completed during a Data Technician bootcamp and focuses on building foundational Python programming skills. Through a range of hands-on exercises and challenges, the project demonstrates how core concepts can be used to create logical, interactive, and calculation-based programs.

---

## 🧠 Skills & Tools Used
- Python (Google Colab / local environment)
- Variables for storing and managing data
- `print()` for output display
- `input()` for collecting user data
- Type casting (`int()`, `float()`, `str()`) for handling data types
- Conditional statements (`if`, `elif`, `else`)
- Looping structures (`for` and `while` loops)

---

## 📊 Data Visualisation Techniques
While this project primarily focuses on programming fundamentals, it lays the groundwork for future data visualisation by:
- Structuring and preparing data through calculations
- Using loops to iterate through datasets
- Organising outputs clearly for readability and interpretation

---

## 📂 Project Tasks
- Created interactive programs that take user input and generate dynamic outputs  
- Built calculation-based scripts (e.g., temperature conversion, area calculations, and number operations) 
- Developed logic-driven solutions such as checking even/odd numbers and validating user input 
- Implemented classic programming challenges like **FizzBuzz** using loops and conditional statements 
- Designed a menu-driven **ATM simulation**, allowing users to check balance, deposit, withdraw, and exit using conditional logic and loops 

---

## 🔍 Key Insights
- Breaking problems into smaller steps improves coding efficiency  
- Combining loops and conditionals enables powerful program control  
- User input handling is essential for building interactive applications  
- Type casting ensures accurate calculations and avoids errors  

---

## 🚀 What I Learned
- How to structure Python programs using logical flow  
- How to combine multiple programming concepts into a single application  
- The importance of validating user input for reliability  
- How repetition (loops) reduces manual coding effort  
- How conditional logic drives decision-making in programs
